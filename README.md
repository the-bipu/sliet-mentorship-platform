## ğŸ“‹ About The Project

SLIET Mentorship Platform is a comprehensive web application designed to bridge the gap between students and faculty members at Sant Longowal Institute of Engineering & Technology (SLIET). The platform provides an intuitive interface for students to discover, connect with, and book mentorship sessions with their preferred professors.

### âœ¨ Key Features

- **Faculty Profiles**: Detailed profiles of SLIET professors with specializations and availability
- **Smart Booking System**: Easy-to-use appointment scheduling interface
- **Category Filtering**: Browse mentors by department and expertise
- **Responsive Design**: Seamless experience across all devices
- **Custom Typography**: Unique Clash Display font for enhanced visual appeal
- **Interactive UI**: Smooth animations and transitions for better user engagement

## ğŸš€ Getting Started

### Prerequisites

No special prerequisites needed! Just a modern web browser.

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/the-bipu/sliet-mentorship-platform.git
   ```

2. Navigate to the project directory
   ```bash
   cd sliet-mentorship-platform
   ```

3. Open `index.html` in your browser
   ```bash
   # Or use a local server (recommended)
   npx serve .
   ```
